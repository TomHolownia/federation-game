# Game Design Documentation

This folder contains game design documentation for the Federation game project.

## Old Documentation

The `old-docs/` folder contains legacy design documents that were converted from the original Word documents and images. These need to be reviewed and vetted before being used in the actual game development.

### Contents of old-docs/

- **Races & Beings** - Race descriptions, skills, class hierarchies
- **Items & Weapons** - All weapon types, armor, consumables, software
- **Planets & Lore** - 34 planets with detailed descriptions and world lore
- **Timeline** - Historical events from 3400 BC to 2300 AD
- **Quests & Events** - Main storylines and side quests
- **Vessels & Systems** - Ships, vehicles, and their components
- **Buildings** - All building types and locations
- **Controls** - Input controls and keybindings
- **Inventory** - UI mockups and equipment system

See `old-docs/README.md` for a full index.
