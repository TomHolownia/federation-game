## Star System Classification

Star systems are classified by their strategic and economic importance:

| Classification | Marker Color | Description |

The Federation territory is located in one of the spiral arms of the galaxy, with star systems connected by established travel routes.

### Notable Features

- Travel routes connect major systems
- Frontier regions extend toward the galactic edge
