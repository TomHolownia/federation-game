## Interface Controls

| Action | Key/Input | Description |
|--------|-----------|-------------|
| Open Inventory | I | Opens inventory with 4 options: Configure, Software, Options, and Save and Quit. Press I again to leave |
| HUD Configure Mode | Hold Ctrl | While in this mode, left click moves objects on the HUD around, and right click edits, changes or deletes them. Press Ctrl to return to normal controls |

## Non-Combat Mode

| Action | Key/Input | Description |
|--------|-----------|-------------|
| Safe Action | Left Click | Includes talking, using consoles, picking up items and sitting down |
| Bad Action | Right Click | Steal, pickpocket, cut wires, jack vehicles |
| Jump | Space | |
| Passive Electronic Scan | Q (Press) | Purely for information, no one can detect a passive scan. Is directed at what you are looking at. Can also be used in combat mode |
| Electronic Warfare | E (Press) | Aggressor software will attack target you are looking at. This can be a computer system or a being. Can be used in combat mode |

### Talking System
Talking works by looking at someone and left clicking. A list of things you want to say comes up, scroll up or down to select one and left click again to speak.

## Quick Select Wheel

To use one of the function slots:
1. Click in the scroll wheel
2. Push mouse in a direction over a slot
3. Release the mouse

### Layout
- The four items on the **left hand side** are for your **left hand**
- The four items on the **right side** are for **right hand**
- Two-handed items can be placed wherever and will be used appropriately
- The **top slot** toggles combat and non-combat modes

### Item Display
- Items with stacks (grenades, smoke bombs, deployable cover balls, combat drugs) will have their number displayed
- Hard wirings will be used instantly and will drain battery
- Weapons will have no number next to them

### Selecting Items
- If you select an aggressive software, you will automatically equip it to your aggressive software slot (press E to use)
- If you select a passive software you will automatically equip it to your passive slot
- Some items will be used immediately (such as combat drugs) and after use your hands will return to whatever they were holding previously
- Some will place an item in your hands like a mine or grenade which you can then place or use
- If you select an offensive slot when in non-combat mode, you will automatically go into combat mode
