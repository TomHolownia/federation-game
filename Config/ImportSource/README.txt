Place source assets here for the Import Assets command (Tools -> Federation -> Import Assets).

Human character (MVP):
  Put one or more human character FBX files in the Human/ subfolder.
  Suggested free sources: Mixamo (mixamo.com), or Epic Marketplace free "Animation Starter Pack" / character packs.
  Then run: Tools -> Federation -> Import Assets.
  Assets are imported to Content/Characters/Human.

Add any third-party assets to the project's CREDITS.md (see repo root).
